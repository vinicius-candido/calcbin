PARA RODAR O PROGRAMA É NECESSARIO JRE 1.8.0 OU SUPERIOR
Para executá-lo importe o projeto java em uma IDE (no desenvolvimento foi utilizado o Eclipse Kepler).
Todo o código está no arquivo \calcbin\src\calcbin\Main.java

Para converter um número decimal em outra base:
 Após executar, tecle "1" e depois <Enter>
 Digite o número decimal que deseja converter e tecle <Enter>
 Digite a base que deseja o número convertido e tecle <Enter>
 Será exibido o número convertido
 
Para converter um número qualquer para a base decimal:
 Após executar, tecle "2" e depois <Enter>
 Digite o número que deseja converter para a base 10 e tecle <Enter>
 Digite a base do número informado e tecle <Enter>
 Será exibido o número convertido
